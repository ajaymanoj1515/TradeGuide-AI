# TradeGuide AI 📈

**TradeGuide AI** is an intelligent stock analysis tool designed to help retail traders automate technical analysis. It eliminates manual chart reading by providing logic-based "Buy", "Sell", or "Hold" signals across multiple timeframes.

## 🚀 Key Features
- **Multi-Timeframe Analysis:** Supports Intraday (15m, 1h) and Swing (Daily, Weekly) trading.
- **Automated Logic:** Instantly calculates RSI (14), MACD, and SMA crossovers.
- **Live Market Data:** Fetches real-time data for NSE (India), US Stocks, and Crypto using Yahoo Finance.
- **Interactive Charts:** Built with Lightweight Charts for a TradingView-like experience.

## 🛠️ Tech Stack
- **Backend:** Python, Flask, Pandas, NumPy
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** SQLite (SQLAlchemy)
